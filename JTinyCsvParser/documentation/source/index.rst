JTinyCsvParser
==============

Welcome to the `JTinyCsvParser`_ documentation!

`JTinyCsvParser`_ is a library to parse CSV data in an easy and fun way, while offering a very 
high performance and a clean API. It is probably the fastest Java CSV Parser around 
(:ref:`Benchmark<benchmark>`) and it is highly configurable to provide maximum flexibility. 

To get started quickly, follow the :ref:`Quickstart <quickstart>`.
